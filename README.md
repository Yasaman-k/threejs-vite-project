# threejs-vue-project
#### This project use three.js library which install in vite.js framework.

```shell
  $ npm install 
  ```
 ```shell
  $ npm run dev
  ```
 
 
- source : https://www.youtube.com/watch?v=Q7AOvWpIVHU 
- source : https://codinhood.com/post/create-skybox-with-threejs

### Setup
install vite.js
  ```shell
  $ npm init @vitejs/app
  ```
  
install tree.js
 ```shell
  $ npm install --save three
  ```
